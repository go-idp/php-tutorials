# 开发者指南

欢迎！如果你已经有其他编程语言的经验，本指南将帮助你快速上手 PHP。

## 选择你的背景

根据你的编程背景，选择对应的学习指南：

### [前端开发者](/guide/for-developers/frontend-developer)

如果你熟悉 JavaScript/TypeScript、HTML、CSS：
- 理解服务器端编程
- 学习 PHP 如何生成 HTML
- 掌握表单处理和 API 开发

### [Node.js 开发者](/guide/for-developers/nodejs-developer)

如果你熟悉 Node.js、Express.js：
- 对比异步 vs 同步执行
- 学习 Composer 包管理
- 理解 PHP 的请求-响应模型

### [Java 开发者](/guide/for-developers/java-developer)

如果你熟悉 Java、Spring：
- 利用 OOP 知识快速上手
- 理解动态类型系统
- 学习 PHP 的自动加载机制

### [Python 开发者](/guide/for-developers/python-developer)

如果你熟悉 Python、Flask/Django：
- 适应花括号语法
- 理解 PHP 数组的灵活性
- 学习 PHP 的 Web 开发特性

### [Go 开发者](/guide/for-developers/go-developer)

如果你熟悉 Go、Gin/Echo：
- 理解动态类型系统
- 学习异常处理模型
- 掌握 PHP 的 Web 开发优势

## 每个指南包含

- **快速对比表**：语法和概念对比
- **共同点**：可以直接迁移的知识
- **主要差异**：需要注意的语法差异
- **快速上手路径**：推荐的学习顺序
- **常见陷阱**：容易出错的地方
- **实战对比**：代码对比示例

## 开始学习

选择你的背景，开始你的 PHP 学习之旅！
