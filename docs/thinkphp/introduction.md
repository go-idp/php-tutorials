# ThinkPHP 框架介绍

ThinkPHP 是一个免费开源的、快速、简单的面向对象的轻量级 PHP 开发框架。

## 什么是 ThinkPHP？

ThinkPHP 是为了简化企业级应用开发和 Web 应用开发而诞生的。它遵循 Apache2 开源协议，使用面向对象的开发结构和 MVC 模式。

## 主要特性

- **简单易用**：学习成本低，文档完善
- **性能卓越**：轻量级框架，运行效率高
- **功能丰富**：内置丰富的功能组件
- **扩展性强**：支持 Composer 扩展
- **社区活跃**：拥有庞大的开发者社区

## 版本说明

- **ThinkPHP 6.x**：最新版本，支持 PHP 7.2+
- **ThinkPHP 5.x**：稳定版本，支持 PHP 5.6+
- **ThinkPHP 3.x**：旧版本，已停止维护

本教程主要基于 ThinkPHP 6.x。

## 适用场景

- Web 应用开发
- API 接口开发
- 后台管理系统
- 企业级应用
