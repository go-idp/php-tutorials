# ThinkPHP 示例项目

这是一个简单的 ThinkPHP 示例项目，展示了基本的 MVC 结构。

## 安装

```bash
composer create-project topthink/think tp
cd tp
```

## 项目结构

```
tp/
├── app/
│   ├── controller/
│   │   └── Index.php
│   └── model/
│       └── User.php
├── config/
│   └── database.php
└── route/
    └── app.php
```

## 使用

参考 ThinkPHP 官方文档进行开发。
