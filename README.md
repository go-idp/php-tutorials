# PHP 从入门到精通教程

全面的 PHP 教程，涵盖基础语法、Web 开发、ThinkPHP 框架和进阶内容。

## 项目结构

```
php-tutorials/
├── docs/              # 文档目录
├── examples/          # 代码示例
└── .github/           # GitHub Actions 配置
```

## 本地开发

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```

### 预览

```bash
pnpm preview
```

## 技术栈

- VitePress - 文档生成器
- TypeScript - 类型支持
- Pnpm - 包管理器

## 部署

项目使用 GitHub Actions 自动部署到 GitHub Pages。
